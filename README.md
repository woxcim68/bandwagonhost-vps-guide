## **搬瓦工到底是什麼？為什麼這麼多人推薦它**

先說說搬瓦工是什麼。搬瓦工，英文叫 BandwagonHost，是 IT7 Networks 旗下的一個 VPS（虛擬專用伺服器）品牌，主打的是「企業級硬體 + 自助管理 + 價格壓到很低」這個組合。它走的是 self-managed 路線，意思是你自己管伺服器，它負責把硬體和網路穩定性做到位，這樣就把人工客服的成本省下來，價格自然就壓下來了。

那為什麼中文圈這麼多人推薦它？幾個原因：

第一，它支援**支付寶付款**。這一點對中國大陸、台灣、香港的用戶來說太關鍵了，不用折騰信用卡、PayPal，掃個碼就完事。

## **搬瓦工註冊教學：從零開始的完整步驟**

### **第一步：進入搬瓦工官網**

打開瀏覽器，訪問搬瓦工官網。這裡有個小提醒：搬瓦工有幾個鏡像域名（比如 bwh88.net、bwh81.net 之類的），都是官方的，功能一樣，選哪個都行。如果你在國內打開主域名有點慢，可以試試鏡像域名。

### **第二步：填寫註冊資訊**

進入註冊頁面後，會看到一個表單。需要填的東西不多，但有一個關鍵點要記住：

**郵箱一定要填真實有效的地址。** 因為搬瓦工的初始密碼、VPS 的 root 密碼、各種通知都會發到這個郵箱。填錯了你就收不到登錄資訊，等於白買。

其他欄位填寫建議：

- **First Name / Last Name**：建議用英文或拼音填，別用中文。倒不是說中文不行，而是有些支付渠道對中文支援不太友好，後續付款可能出問題。圖省事就用拼音。
- **Country**：選你所在的國家/地區就行。
- **Email**：重點，前面說過了。
- **Password**：自己設一個記得住的。
- **Security PIN**：會讓你設一個 4 位數的 PIN 碼，這個用於後續登錄 KiwiVM 控制面板時的二次驗證，記好它。

填完之後，勾選同意條款，點「Register」提交。這時候系統會發一封驗證郵件到你填的郵箱，去郵箱裡點確認連結，賬號就激活了。

### **第三步：選擇套餐並加入購物車**

賬號註冊好之後，接下來就是挑套餐。搬瓦工的套餐分幾大類：

- **普通 KVM 套餐**：走美國常規線路，價格最便宜，適合預算有限、對延遲不敏感的場景
- **CN2 GIA-E 套餐**：走中國電信 CN2 GIA 高端線路，三網優化，晚高峰也不堵，性價比最平衡
- **香港 / 日本 / 新加坡 CN2 GIA 套餐**：亞洲頂級線路，延遲最低，但價格也最貴
- **迪拜 E-Commerce 套餐**：中東節點，覆蓋中東、歐洲、非洲用戶

具體怎麼選，後面的套餐對比部分會詳細講。現在先假設你已經挑好了，點對應套餐的「Order」按鈕，進入配置頁面。

### **第四步：選擇計費週期和機房**

進入配置頁面後，有兩個東西要選：

**計費週期**：搬瓦工的套餐支援月付、季付、半年付、年付，週期越長單月均價越便宜。比如 20G 普通套餐，年付 $49.99，算下來一個月才 4 美元出頭；要是選月付就貴很多。所以一般推薦直接選年付，省錢。

**機房位置**：不同套餐可選的機房不一樣。普通套餐可以選美國多個機房（DC2、DC4、DC8 等）和荷蘭機房；CN2 GIA-E 套餐可以選洛杉磯 DC6/DC9、聖何塞、紐約、溫哥華等多個機房，而且買完之後還能免費在這些機房之間隨便切換。

選好之後，點「Add to Cart」加入購物車。

### **第六步：結賬並付款**

優惠碼生效後，點「Checkout」進入結賬頁面。如果你還沒登錄，系統會讓你登錄或註冊（這就是前面說的「順便註冊」的路徑）。

確認訂單資訊沒問題後，選擇支付方式。搬瓦工目前支援的支付方式有：

- **支付寶（Alipay）**：國內用戶首選，掃碼付款，人民幣自動換算成美元，方便快捷
- **銀聯（UnionPay）**：儲蓄卡、信用卡都行，跳轉到銀聯收銀台
- **PayPal**：有 PayPal 賬號的都能用
- **信用卡**：Visa、MasterCard 等

選好支付方式後，點「Complete Order」，跳轉到支付頁面。用支付寶的話會出來一個二維碼，手機掃一下就完事了。

付款成功後，搬瓦工會把 VPS 的登錄資訊（IP、root 密碼、KiwiVM 面板地址）發到你註冊時填的郵箱。一般幾分鐘內就能收到，如果超過 10 分鐘還沒收到，先去垃圾郵件夾翻翻，再沒有就檢查下註冊郵箱是不是填錯了。

**Q：CN2 GIA 和 CN2 GIA-E 有什麼區別？**
